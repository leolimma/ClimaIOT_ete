Sobre o Sistema

Objetivo

Monitorar em tempo real as condições climáticas da Escola Técnica Estadual Pedro Leão Leal, com coleta automática de dados via IoT e disponibilização através de interface web intuitiva.

Funcionalidades

  • Monitoramento em tempo real
  • Histórico de dados
  • Exportação de dados (CSV/PDF)
  • Relatórios personalizados
  • Painéis administrativos
